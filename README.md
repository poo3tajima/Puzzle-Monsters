# name
Puzzle & Monsters

## Overview
Pythonで作成したテキストベースのバトルゲームです。
入門書「スッキリわかるPython入門」にある練習課題を作りました。
ゲームバランスを考えて少しアレンジしてあります。

主人公は４体の聖獣を率いてダンジョンへ入り、５体のモンスターを倒します。
主人公たちはモンスター１体と順番に戦い、順番に攻撃しあいます。
コマンドを入力し、宝石を動かして戦います。
宝石が３つ以上そろうとその宝石に応じた聖獣がモンスターを攻撃します。
順番に倒して進み、５体目のドラゴンを倒せばゲームクリアです。
聖獣のＨＰが全員０になったら、ゲームオーバーです。

## Requirement
Windows用です。

＜Windows10の場合＞
コマンドプロンプトで起動すると、色コードが表示されないのでプレイが出来ません。
Microsoft Storeで「 Windows Terminal 」をインストールしてください。
exeを一度起動し、タイトルバーを右クリックして「設定」を選びます。
スタートアップタブの一番上の項目の「既定のプロファイル」を「Windows Power Shell」に変更してください。
再度exeを起動するとPower Shellで色コードが表示されるようになります。

## Usage
puzmon.exeをダウンロードして、実行してください。
警告画面が出たら、文章の中にある「詳細表示」を押すと、実行を選べるようになります。

ウインドウをタテに大きくひろげてプレイするとやりやすいです。
Ctrl + C でプログラムを中断出来ます。
