# name
Puzzle & Monsters　　パズル ＆ モンスターズ

## Overview
Pythonで作成したテキストベースのバトルゲームです。<br>
入門書「スッキリわかるPython入門」にある練習課題を作りました。<br>
ゲームバランスを考えて少しアレンジしてあります。

主人公は４体の聖獣を率いてダンジョンへ入り、５体のモンスターを倒します。<br>
主人公たちはモンスター１体と順番に戦い、お互いに攻撃しあいます。<br>
コマンドを入力し、宝石を動かして戦います。<br>
宝石が３つ以上そろうとその宝石に応じた聖獣がモンスターを攻撃します。<br>
敵味方お互いの属性で効果が変わります。<br>
順番に倒して進み、５体目のドラゴンを倒せばゲームクリアです。<br>
聖獣のＨＰが全員０になったら、ゲームオーバーです。

## Requirement
WindowsとMacでプレイできます。

＜Windows10の場合＞<br>
コマンドプロンプトで起動すると、文字に色が付かないのでプレイが出来ません。<br>
Microsoft Storeで「 Windows Terminal 」をインストールしてください。<br>
exeを一度起動し、タイトルバーを右クリックして「設定」を選びます。<br>
スタートアップタブの一番上の項目の「既定のプロファイル」を「Windows Power Shell」に変更してください。<br>
再度exeを起動するとPower Shellで起ち上がり、文字に色が付いてプレイ出来ます。

## Usage
＜ダウンロード＞<br>
puzmon_win.exe<br>
puzmon_mac<br>
をクリックして画面が変わったら、右にある【・・・】アイコンをクリックして、<br>
ダウンロードを選んでください。

Windowsで実行したときに警告画面が出たら、文章の中にある「詳細表示」を押すと、実行を選べるようになります。<br>
Windowsのコマンドプロンプトで文字サイズを変更するにはCtrl + ＋(－)で出来ます。<br>
Macのターミナルで文字サイズを変更するには⌘ + ＋(－)で出来ます。<br>
ウインドウをタテに大きくひろげてプレイするとやりやすいです。<br>
Ctrl + C でプログラムを中断出来ます。
